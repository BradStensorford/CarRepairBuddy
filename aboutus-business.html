Car Repair Buddy

A software project aimed in helping people in need of technical assistance. This project is a collaborative work inorder to enhance the experience of people who are troubleshooting car problems.


Program version: 1.3 beta
This project made use of the following components:
  • qgis2web
  • Open Layers
  • Jquery with Jquery UI
  • OpenStreet Maps
  • PhoneGap

Thank you for using this software and this would not be possible if it were not for the never-ending support of my friends, classmates and family.
